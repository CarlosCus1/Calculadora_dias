# Calculadora de Días y Montos

Una aplicación web de una sola página (SPA) diseñada para facilitar la planificación financiera y la gestión de fechas, especialmente útil para el cálculo de letras anticipadas de campaña.


*(Aquí puedes añadir una captura de pantalla de tu aplicación)*

## ✨ Características Principales

- **📅 Doble Calendario Interactivo:** Selecciona fechas de manera visual e intuitiva en dos meses consecutivos.
- **⏳ Cálculo de Días Restantes:** Muestra automáticamente los días que faltan para cada fecha seleccionada.
- **🇵🇪 Feriados y Domingos:** Resalta automáticamente los domingos y los feriados peruanos para una mejor planificación.
- **📋 Portapapeles Inteligente:** Copia las fechas seleccionadas en un formato optimizado, listo para pegar.
- **💵 Distribuidor de Montos:** Ingresa un monto total y distribúyelo equitativamente entre las fechas elegidas.
- **📊 Reportes Detallados:**
    - **Resumen Mensual:** Visualiza los totales y porcentajes por mes.
    - **Tabla Detallada:** Ajusta los montos individuales por fecha y ve los cambios reflejados en tiempo real.
- **⬇️ Exportación a Excel:** Descarga un reporte completo y profesional en formato `.xlsx` con un solo clic.
- **🎨 Tema Oscuro/Claro:** Cambia entre modos de visualización para mayor comodidad.
- **📱 Diseño Responsivo:** Totalmente funcional en dispositivos de escritorio y móviles.
- **♿ Accesibilidad:** Interfaz mejorada con modales personalizados y etiquetas ARIA para una mejor experiencia con lectores de pantalla.

## 🚀 Tecnologías Utilizadas

- **HTML5**
- **CSS3:** Uso de variables CSS para theming y diseño responsivo con Flexbox y Grid.
- **JavaScript (ES6+):** Lógica de la aplicación organizada en Clases y Módulos, sin dependencias de frameworks.
- **SheetJS (xlsx):** Para la generación y descarga de archivos de Excel.
- **Font Awesome:** Para la iconografía de la interfaz.

## ⚙️ Uso

Simplemente abre el archivo `index.html` en tu navegador web preferido. No se requiere instalación ni configuración de servidor.
